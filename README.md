# Application_Form_Project_using_react
------------------------------------------------------------------------------------------
This is a Front-End Application Form

## Features
A. Add Appointment  
B. Delete Existing Appointment  
C. Search for an appointment    
D. Sort the existing entries in ascending / descending order of    
 - Date of Appointment    
 -  First Name  
 -  Last Name
## Technologies Used

    1. HTML5

    2. CSS3

    3. Javascript ES6

    4. Bootstrap 5

    5. ReactJS

    6. React Routers
